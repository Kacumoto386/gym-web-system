# 开发日志 V3.2.8
## 课程包管理（统一管理页+双Tab+模型扩展）

**日期**: 2026-05-12 17:16
**版本**: v3.2.7 → v3.2.8

### 新增功能

统一「课程包管理」页面 `/packages`，双 Tab 结构：

**Tab 1: 📦 产品管理**
- 团课打包产品（`group_package`）的 CRUD
- 2 种类型：🎯 计次打包 / ♾️ 不限次数
- 包含课程多选（关联 `course` 表）
- 上架/下架快捷切换

**Tab 2: 🛒 已售课程包**
- 合并展示 3 种会员已购资源：
  - 📦 普通课包（`lesson_package`，`package_type=normal`）
  - 🎯 团课计次包（`lesson_package`，`package_type=group`）
  - ♾️ 包月团课 / 包月私教（`monthly_pass`，`pass_type=group/private`）

### 模型变更

| 表 | 变更 | 说明 |
|----|------|------|
| `lesson_package` | 新增 3 列 | `package_type`(normal/group) / `included_courses` / `course_names` |
| `monthly_pass` | 新增 1 列 | `pass_type`(group=包月团课/private=包月私教) |

### 新增 API 端点

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/packages/products/table` | 产品列表 HTML 片段 |
| POST | `/api/packages/products` | 新建产品 |
| PUT | `/api/packages/products/{id}` | 编辑产品 |
| POST | `/api/packages/products/{id}/toggle-status` | 上架/下架切换 |
| DELETE | `/api/packages/products/{id}` | 删除产品 |
| GET | `/api/packages/member-packages?keyword=&status=` | 已售课包列表 HTML 片段 |
| GET | `/api/packages/courses-options` | 课程多选下拉 HTML |
| GET | `/api/packages/products/{id}` | 产品详情（编辑用） |
