# 开发日志

## 2026-05-09 18:55 - V3.2.0 摄像头拍照功能

### 需求
会员管理编辑弹窗已有本地上传照片功能，需要增加摄像头拍照上传。

### 实现方案
纯前端实现，后端零改动：
1. 照片区增加"📸 拍照"按钮
2. 点击弹出摄像头弹窗（模态框），浏览器 `getUserMedia` 实时预览
3. 点击拍照 → `canvas.drawImage()` 截取帧 → `canvas.toBlob()` → FormData → `POST /api/members/{id}/photo`
4. 上传成功自动刷新编辑弹窗照片，2秒后关闭弹窗
5. 支持重拍、关闭时自动停止摄像头流释放资源

### 异常处理
- 无摄像头设备 → 显示提示
- 权限被拒绝 → 显示引导
- 上传失败 → 显示错误，可重拍
