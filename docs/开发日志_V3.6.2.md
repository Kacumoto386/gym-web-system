# 开发日志 V3.6.2 — 会员管理页面 UI 紧凑化

**日期**: 2026-05-22  
**类型**: refactor

---

## 变更内容

### 1. 标题栏

- 标题下边距 `mb-4` → `mb-3`
- 新增会员按钮 `px-4 py-2` → `px-3 py-1.5`

### 2. 筛选栏

- 容器内边距 `p-4` → `p-3`，下边距 `mb-4` → `mb-3`
- 两行间距 `gap-3` → `gap-2`，行间距 `mb-3` → `mb-1.5`
- 搜索框/下拉框/日期输入 `px-3 py-2` → `px-2 py-1.5`
- 查询/重置按钮 `px-4 py-2` → `px-3 py-1.5`
- 快捷日期按钮 `px-3 py-1.5` → `px-2 py-1`
- 搜索图标 `top-2.5` → `top-2`，搜索框右边距 `pr-3` → `pr-2`

### 3. 统计卡片

- 卡片内边距 `p-3` → `p-2.5`
- 网格间距 `gap-3` → `gap-2`
- 容器下边距 `mb-5` → `mb-3`
- 数值上边距 `mt-1` → `mt-0`

### 4. 表格

- 表头/数据单元格 `px-4 py-3` → `px-3 py-2`
- 等级/状态徽章 `px-2 py-0.5` → `px-1.5 py-0.5`

### 5. 分页

- 上边距 `mt-4 pt-3` → `mt-3 pt-2`

### 6. 模态框（新增/编辑/拍照/售卡）

- 模态框内边距 `p-6` → `p-5`
- 表单项网格间距 `gap-4` → `gap-3`
- 标题下边距 `mb-4` → `mb-3`
- 新增模态框按钮上边距 `mt-6` → `mt-4`

### 7. 加载占位

- 统计卡片加载占位 `py-4` → `py-2`
- 表格加载占位 `py-8` → `py-4`
