# 开发日志 V3.7.4 — 操作日志覆盖 + 侧边栏折叠 + JS 抽取

**日期**: 2026-05-22  
**类型**: refactor, feature

---

## 变更内容

### 1. 操作日志覆盖不完整修复

- 商品零售编辑、上课记录更新等遗漏的写操作现已自动记录
- `_KNOWN_ACTION_SEGMENTS` 排除非 ID 路径段，避免 resource_id 误提取
- `_RESOURCE_CN_MAP` 扩充覆盖所有业务模块的中文映射

### 2. 侧边栏分组折叠

- 菜单分组（会员/课程/进场/员工/财务/系统）支持点击展开/收起
- 使用 `localStorage` 持久化折叠状态
- 当前页面所在分组自动展开
- 箭头图标 `▼`/`▶` 指示状态

### 3. 侧边栏菜单顺序优化

- 菜单按业务流重新排序：首页 → 会员 → 课程 → 进场 → 员工 → 财务 → 系统
- 子菜单逻辑分组，层次清晰

### 4. JS 抽取到独立文件

- 将 25+ HTML 模板中的内联 JS 逐步抽取到独立文件
- 公共 JS 集中在 `frontend/static/js/app.js`
- 页面特有 JS 通过 `{% block page_js %}{% endblock %}` 加载
- 模板只保留事件绑定和初始化调用
- 总减少内联 JS 约 5000+ 行

### 5. 基础模板重构

- `base.html` 侧边栏改为动态折叠结构
- 导航链接统一使用模板条件判断实现高亮
- 全局弹窗（作废弹窗等）移至 base.html
