# 开发日志 V3.6.4 — 跟进员工筛选修复 + favicon 路由

**日期**: 2026-05-22  
**类型**: fix, chore

---

## 变更内容

### 1. 跟进员工筛选举证不到数据

**现象**: 筛选栏选择"李亮"等员工后表格返回空数据。

**原因**: 历史数据中 1730/1732 条会员的 `staff_id` 为空，1723 条仅有 `staff_name`（存的是员工姓名）。原来的 `Member.staff_id == staff_id` 过滤条件无法匹配。

**修复**: `member_table()` 中传入 `staff_id` 时先查 Staff 表获取对应姓名，然后用 `OR` 条件同时匹配 `Member.staff_id` 和 `Member.staff_name`。

### 2. 新增 `/favicon.ico` 路由

浏览器自动请求 `/favicon.ico` 返回 404，添加路由重定向到 `/static/favicon.svg` 消除报错。
