## 变更内容

### 售课记录编辑功能

售课记录列表页的"编辑"按钮之前仅为 `alert('编辑功能开发中')`，现实现完整编辑功能。

#### 后端 (`backend/routers/sale.py`)

1. **新增 `SaleDetail` 模型** — 继承 `from_attributes = True`，返回全部 26 个字段供编辑弹窗预填
2. **GET /api/sales/{sale_id}** — 响应模型 `SaleOut` → `SaleDetail`，编辑弹窗能获取完整数据
3. **PUT /api/sales/{sale_id}** — 去掉 `if val is not None` 守卫（允许清空 start_date/end_date）；新增 `sale.total_hours = sale.bought_hours + sale.bonus_hours` 重算总课时
4. **编辑按钮** — `_build_table()` 中 `alert(...)` → `openEditSale('{s.sale_id}')`

#### 前端 (`frontend/templates/sales.html`)

1. **编辑模态框** — 独立于新增弹窗，id 使用 edit 前缀避免冲突；含会员/课程/员工搜索下拉、数值字段、支付方式/状态选择、日期字段
2. **JS 函数** — `openEditSale()` 加载数据预填表单、`submitEditSale()` 收集表单 PUT 提交、搜索下拉全套函数（edit 前缀）

## 验证方法

1. 售课记录列表点击"编辑" → 弹窗打开，所有字段预填正确
2. 修改任意字段（如实收金额、支付方式）→ 保存 → 表格更新
3. 清空开卡/到期日期 → 保存 → 再次编辑确认已清空
4. 修改购买课时 → 保存 → 总课时自动重算
5. 关闭弹窗不保存 → 无变化
