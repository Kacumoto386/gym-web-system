## 变更内容

### 后端
- **脏数据过滤**: 所有查询增加 `member_id != '-'` 过滤（9条会籍卡关联无效会员）
- **课程包单价改进**: `calc_sale_unit_price` 支持逗号分隔的多课程 ID（如 "C005,C008"），多条记录取平均价，多课程组合 Fallback 到 Course 标准价取平均
- **Course.name 字段修正**: `get_course_name()` 从 DB `name` 字段获取课程名
- **主表格新增状态列**: 显示各会员资产整体状态标签（有效/即将到期/已过期）
- **汇总卡片新增资产健康**: 第6张卡片显示即将到期/已过期项数
- **CSV导出增加状态列**: 新增"资产状态"列

### 前端
- **状态标签**: 每项资产显示有效/即将到期/30天内到期/已过期标签
- **课时进度条**: 私教课显示已用/共/剩余占比进度条
- **明细弹窗标题**: 显示会员名称而非编号

### Bug修复
| Bug | 原因 | 修复 |
|-----|------|------|
| 会籍卡残值跳脏数据 | `member_id='-'` 未过滤 | 增加 member_id 为空/'-' 跳过 |
| 团课包单价为0 | course_id 含逗号无法匹配 | 用 in_ 查询并取平均 |
| 表格花括号 | `build_assets_table()` 中 `str.join()` 误用 | 改为 `\"\".join(trs)` |
| 全量查询少会员 | member_id='-' 脏数据排除不完整 | 增加完整过滤 |
