# 开发日志

## 2026-05-09 18:18 - V3.1.8-hotfix 期限卡签到 + 现金卡自定义扣费

### 问题 1：期限卡签到选项不可见

**现象**：选择期限卡后，核销方式中没有"期限卡签到"选项。

**根因**：数据库 `membership_card.card_type` 值为"期限卡"，但前端 JS 用 `['月卡','季卡','年卡','时卡'].includes(ctype)` 判断，`ctype='期限卡'` 永远无法匹配。

**修复**：3 处条件判断增加 `ctype === '期限卡'` 分支。

### 问题 2：现金卡扣费不能自定义金额

**现象**：现金卡扣费默认 ¥30，没有自定义输入框。

**修复**：前端去掉 `value="30"` 改为 `placeholder="输入金额"`；后端改为 `consume_amount` 不可为空且必须 > 0。

### 问题 3：浏览器缓存导致新版 JS 不生效

**修复**：路由响应添加 `Cache-Control: no-cache` 头。
