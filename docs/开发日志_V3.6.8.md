# 开发日志 V3.6.8 — 收入支出报表功能扩展

**日期**: 2026-05-22  
**类型**: feature

---

## 变更内容

### 1. 编辑功能

收入支出报表新增编辑功能，参考 `sale.py` 的 `get_sale`/`update_sale` 模式：
- 后端新增 `IncomeUpdate`/`ExpenseUpdate` Pydantic 模型
- `GET /api/finance/income/{record_id}` 返回单条记录 JSON（编辑预填）
- `PUT /api/finance/income/{record_id}` 更新记录，`exclude_unset=True` 局部更新
- 支出同理

前端新增编辑弹窗，`openEditIncome(id)` → fetch 预填 → `submitEditIncome()` PUT 提交 → 关闭弹窗刷新表格。

### 2. 分页 + 搜索 + 筛选

- 定义 `PAGE_SIZE = 50`
- 参数：`page`、`q`（跨 source/payee 和 remark 字段搜索）、`category`
- 总记录数查询 `query.count()` + 计算 `total_pages`
- `_build_pagination()` HTML（复制 `member.py` 的分页函数）
- 搜索关键词、分类筛选、分页联动

### 3. CSV 导出

`GET /api/finance/export?type=income|expense&year=&month=&q=&category=`
- `StreamingResponse` + `csv.writer` + UTF-8 BOM
- 中文表头，Content-Disposition 下载

### 4. 分类扩展

- **收入分类**: 售课收入、商品零售、会籍卡、私教课程、团课收入、储物柜租金、其他
- **支出分类（15种）**: 房租、物业水电、工资、设备、办公用品、快递费、打印费、通讯费、差旅费、招待费、维修费、网络费、保险费、税费、其他
- **支付方式（6种）**: 现金、微信、支付宝、转账、刷卡、对公账户
