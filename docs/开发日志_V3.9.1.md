# 开发日志 V3.9.1 — 员工端小程序后端修复 + 版号 S1.0.1

**日期**: 2026-05-27  
**类型**: fix

---

## 变更内容

### 1. 员工端后端 7 模块修复（26+ 处字段名不匹配）

| 模块 | 文件 | 修复内容 |
|------|------|----------|
| 卡种销售 | card_sale.py | Product→CardProduct, 构造参数全部修正 + 自动记录 Sale |
| 课程销售 | course_sale.py | 构造参数全部修正 + 添加 sale_date |
| 预约管理 | booking.py | role→position, staff_id→coach_id, staff_name→coach_name |
| 签到核销 | checkin.py | consume_note→consume_detail, 日期时间分离, 排序修正 |
| 会员查询 | member.py | product_name→card_name（响应 key 保留 product_name 兼容） |
| 业绩统计 | performance.py | salesperson→staff_name, paid_amount→actual_amount |
| 上课记录 | class_record.py | 补充 record_id 生成 + notes→remark 映射 |

### 2. 数据库迁移
- MembershipCard 新增 `remaining_classes` 字段（ALTER TABLE）

### 3. 员工端小程序版号 S1.0.1
- app.js 新增 `version: 'S1.0.1'`
- 首页底部显示 "员工小程序 vS1.0.1"
