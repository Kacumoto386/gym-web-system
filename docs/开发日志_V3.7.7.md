**类型**: fix

## 变更内容

### 1. 修复 MCP 数据查询工具静默崩溃 bug

**根因**：`backend/mcp/business_tools.py` 中 `_json_safe()` 函数调用 `json.dumps()` 但文件未导入 `json` 模块，导致 `list_members`、`get_member`、`list_staff`、`list_courses`、`list_sales`、`get_finance_summary`、`get_dashboard_stats`、`search` 等所有 MCP 数据查询工具均静默返回 `NameError: name 'json' is not defined`。AI 助手拿到错误后只能回复"系统故障"。

**修复**：第 10 行添加 `import json`。

### 2. 修复 conversation.py 缺失 logging

`conversation.py:427` 调用 `log.warning()` 但 `log` 从未定义。添加 `import logging` 和 `log = logging.getLogger(__name__)`。
