## 2. 改造内容

### 2.1 表头点击排序
为 `self.tree` 的每列添加 `command` 回调，点击表头按该列排序。

| 列名 | 排序类型 | 数据类型 |
|------|---------|---------|
| 课程包编号 | 文本 | str |
| **会员姓名** | 文本 | str ✅ **指定优化** |
| 课程名称 | 文本 | str |
| 总课时 | 数字 | int |
| 已消耗 | 数字 | int |
| **剩余课时** | 数字 | int ✅ **指定优化** |
| 消耗率 | 数字 | float |
| **有效期起** | 日期 | datetime ✅ **指定优化** |
| **有效期止** | 日期 | datetime ✅ **指定优化** |
| 状态 | 文本 | str |

### 2.2 智能排序（根据字段类型）
- **文本列**（会员姓名/课程名称/编号/状态）：`str(key)` 字典序
- **数字列**（总课时/已消耗/剩余课时/消耗率）：按 `int/float` 排序
- **日期列**（有效期起/止）：按 `datetime` 排序

> 不依赖外部库，纯 ttk.Treeview `heading(command=...)` 实现。

### 2.3 排序方向指示器
- 升序/降序交替切换（单击一次升序，再单击降序）
- 在表头文本后添加 ▲/▼ 符号（如 `有效期起 ▲`）

### 2.4 排序状态保持
- 排序后不影响筛选
- 刷新后重置排序状态
- 记录当前排序列和方向

## 3. 实现方案

### 方案 A：利用 `base_frame.py` 已有排序（推荐）
`base_frame.py` 已有 `_sort_by()` 方法，但它的实现是：
1. 纯文本排序（不适合数字/日期）
2. 只升序不能降序切换

### 方案 B：独立实现智能排序（更优）
在 `PerformancePackageFrame` 中直接实现增强版排序：

```python
def _sort_tree(self, col_name, reverse=False):
    """按列智能排序，支持数字/日期/文本"""
    items = [(self.tree.set(child, col_name), child) for child in self.tree.get_children("")]
    
    # 智能解析值
    def sort_key(val):
        val = str(val).strip()
        # 日期匹配 YYYY-MM-DD
        if re.match(r'^\d{4}-\d{2}-\d{2}$', val):
            return (0, val)  # 日期优先
        # 数字（含 % 后缀）
        try:
            num = float(val.replace('%', ''))
            return (1, num)
        except ValueError:
            pass
        # 纯数字字符串（如 "120"）
        try:
            return (1, int(val))
        except ValueError:
            pass
        return (2, val)
    
    items.sort(key=lambda x: sort_key(x[0]), reverse=reverse)
    for idx, (_, child) in enumerate(items):
        self.tree.move(child, "", idx)
```
