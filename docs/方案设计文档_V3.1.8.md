# V3.1.8 首页重设计 + 进场核销升级

**日期**: 2026-05-09
**仓库**: projects/gym-web-system
**备份**: `projects/_backups/gym-web-system_v3.1.8_备份_20260509_143631/` (90 files, 1333 KB)

## 核心变更

### 1. 首页重设计
- **快捷入口按钮缩小**: 从 `p-5` 缩小到 `p-3`，从 `grid-cols-2/sm:grid-cols-4` 改为 `grid-cols-3/sm:grid-cols-4/md:grid-cols-6`
- **新增今日进场记录区块** (左侧2/3宽度):
  - 顶部: 标题 + 今日总人数 + "管理进场"链接
  - 统计行: 核销入场 | 刷卡入场 | 无卡体验 三栏数字
  - 最近记录列表: 头像、姓名、手机号、核销方式标签
- **新增快速签到入口** (右侧1/3宽度):
  - 输入框: 手机号 / 会员编号，300ms 防抖搜索
  - 自动展示会员信息（卡类型、剩余次数、余额）
  - 核销方式选择: 次卡扣次 / 储值扣款 / 期限卡签到 / 无卡体验
  - 一键签到，签到后自动刷新左侧今日进场列表

### 2. 前端加载优化
- **本地化 JS 库**: 不再依赖 CDN，全部改为 `/static/` 本地加载
  - `tailwind.min.js` (407 KB) — 从 cdn.tailwindcss.com 下载
  - `htmx.min.js` (51 KB) — 从 unpkg.com 下载
- **FastAPI 静态文件挂载**: 新增 `StaticFiles` mount 提供 `/static/` 路径

### 3. 后端新增 API
| 端点 | 方法 | 说明 |
|------|------|------|
| `/api/dashboard/today-checkins` | GET | 今日进场记录列表+按方式统计（HTML 片段） |

### 4. 认证中间件优化
- 新增 `PUBLIC_API_PREFIXES` 配置
- `/api/dashboard/` 和 `/api/members/search-json` 免登录访问
