## 变更内容

### 1. AI 助手浮动对话窗口

在首页`index.html`右下角添加浮动 AI 助手对话窗口，提升 AI 功能曝光率和利用率。

**交互逻辑：**
- 右下角圆形 🤖 按钮（64px），点击展开/关闭对话窗口
- 对话框 480×640px，可滚动消息区 + 底部输入栏
- 关闭按钮在右上角，按钮变为 ❌ 红色旋转状态
- 移动端自动适配为全屏底部弹出（80vh）

**功能特性：**
- 欢迎语 + 快捷提问标签（今日进场/本月营收/查会员/课程列表/教练排班）
- 输入问题后调用 `/api/chat/message` API 获取 AI 回复
- 内置简易 Markdown 渲染器，支持表格/代码块/粗体/链接/引用/列表
- "加载中" 跳动圆点动画
- 错误提示（红色文字）

**技术实现：**
- 全部使用内联 CSS + JS（无外部依赖，与已有的 Tailwind + HTMX 不冲突）
- 使用 `fetch` + POST JSON 调用后端 API，非 HTMX
- 对话历史不清空，增强连贯性（调用时自动移除快捷标签）
- 会话 ID 固定为 `widget`

### 2. 版本号更新

- `app.py`: V3.7.5 → V3.7.6
- `base.html`: 侧边栏和底部版本号同步更新
