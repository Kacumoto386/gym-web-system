# 开发日志 V3.7.5 — 作废/红冲机制 + UI 优化

**日期**: 2026-05-22  
**类型**: feature, ui

---

## 变更内容

### 1. 作废/红冲机制

7 个核心模型增加作废字段（`voided`, `void_reason`, `void_time`, `void_operator`）：
- Sale（售课记录）
- Recharge（充值记录）
- ProductSale（商品零售）
- MembershipCard（会籍卡 - 已售卡部分）
- ClassRecord（上课记录）
- FinanceIncome（收入记录）
- FinanceExpense（支出记录）

每个模型对应的路由新增 `PUT /{id}/void` 端点：
- 标记作废标志 + 记录原因/时间/操作人
- 自动处理业务副作用：充值作废反向扣减会员余额、零售作废归还库存
- 已作废记录在列表中半透明显示（`opacity-50`）+ "已作废" 标签
- 主列表查询过滤 `voided == 0`，新增 `/voided/table` 端点查看已作废记录
- 操作人从 JWT token 提取

### 2. 全局作废弹窗

- `base.html` 新增共享的作废弹窗（modal），使用 `bg-black/40` 遮罩
- `app.js` 新增 `openVoidModal/closeVoidModal/confirmVoid` 函数
- 作废原因必填校验，`PUT` 请求提交作废

### 3. 会籍卡菜单迁移

- 侧边栏"会籍卡"链接从"财务"分组移至"会员"分组

### 4. 教练排班日历 UI 美化

- 日期数字改为圆形背景（`w-10 h-10` 40px，`rounded-full`）
- 今天日期：蓝色实心圆（`bg-blue-600 text-white shadow-md`）
- 普通日期：浅灰背景（`bg-gray-100 text-gray-700`）
- 单元格高度统一（`min-h-[80px]`）
- 状态圆点居中显示
- 星期头文字加粗加宽（`tracking-wider`）
