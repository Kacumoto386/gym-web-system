# 开发日志 V3.6.3 — 会员管理页面表格列宽优化 + 分页截断

**日期**: 2026-05-22  
**类型**: refactor

---

## 变更内容

### 1. 统计卡片重构

四个统计卡片由独立卡片改为紧凑的 **inline-flex 药丸条**，标签/数值/单位同行显示，高度从 ~80px 降至 ~28px。

### 2. 筛选栏紧凑化

- 搜索框 `flex-1` → 固定 `w-52`，不再弹性拉伸
- 日期输入 `w-36` → `w-24`，标签缩短
- 快捷按钮 `px-2 py-1` → `px-1.5 py-0.5`，第二行间距 `gap-2` → `gap-1.5`

### 3. 表格列宽固定百分比

每列指定固定宽度百分比（总和 100%），防止短列过宽、长列过窄：

| 列 | 宽度 |
|---|---|
| 编号 | 7% |
| 姓名 | 10% |
| 性别 | 4% |
| 手机号 | 13% |
| 等级 | 7% |
| 剩余课时 | 5% |
| 储值余额 | 9% |
| 最近签到 | 10% |
| 来源 | 6% |
| 跟进 | 6% |
| 状态 | 7% |
| 操作 | 16% |

### 4. 分页截断

超过 10 页时折叠为 `1 … [prev] [当前] [next] … N`，不再显示全部页码。
