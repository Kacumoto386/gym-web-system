## BUG 修复清单

### BUG-1: `response_model=List` 裸泛型导致序列化失败
| 项目 | 内容 |
|------|------|
| **文件** | `backend/routers/finance.py`, `backend/routers/membership_card.py` |
| **端点** | `GET /api/finance/income`, `GET /api/finance/expense`, `GET /api/membership-cards` |
| **错误** | `Unable to serialize unknown type: <class 'backend.models.models.FinanceIncome'>` |
| **修复** | 移除 `response_model=List`，让 FastAPI 使用 `jsonable_encoder` 自动转换 |

### BUG-2: BodyMeasurementOut 必填字段遇数据库 NULL
| 项目 | 内容 |
|------|------|
| **文件** | `backend/routers/body_measurement.py` |
| **端点** | `GET /api/body-measurements` |
| **错误** | `measure_date: Input should be a valid date, input: None` |
| **修复** | `measure_date: date` → `measure_date: Optional[date] = None` |

### BUG-3: commission 缺少 GET /tiers 端点
| 项目 | 内容 |
|------|------|
| **文件** | `backend/routers/commission.py` |
| **端点** | `GET /api/commission/tiers` → 405 |
| **修复** | 新增 `@router.get("/tiers")` 端点，返回与 `/tiers/list` 相同数据 |

### BUG-4: chat_router.py Python 转义序列警告
| 项目 | 内容 |
|------|------|
| **文件** | `backend/routers/chat_router.py:423-424` |
| **警告** | `SyntaxWarning: invalid escape sequence '\/'` |
| **修复** | JS 正则中 `\/` 改为 `\\/`（Python 字面量中正确转义） |

### BUG-5: 测试脚本 form/json 混用
| 项目 | 内容 |
|------|------|
| **文件** | `test_stability.py` |
| **问题** | staff/course 创建用 `data=`（form）但端点期望 JSON（Pydantic 模型） |
| **修复** | 改为 `json=` 发送 |
