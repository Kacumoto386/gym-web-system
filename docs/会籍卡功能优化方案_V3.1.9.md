# 会籍卡功能优化方案 V3.1.9

## 一、概述

对现有会籍卡管理模块进行三项优化：

| # | 优化项 | 说明 |
|:-:|--------|------|
| 1 | **卡产品设置独立页面** | 从会籍卡页面拆出"卡产品"管理，支持独立的 CRUD 页面，保留与已售卡的关联 |
| 2 | **现金卡面值 + 售价分离** | 新增"面值"字段（仅展示，不关联售价），售价归为储值实际金额 |
| 3 | **次卡赠送次数** | 售卡时可选赠送次数，已售数据展示赠送次数 |

---

## 二、卡产品独立设置页面

### 2.1 现状

目前"卡产品"（即产品模板）和"已售会籍卡"混在同一个页面，通过 is_product=1/0 区分。卡产品侧边栏没有独立入口。

### 2.2 改造目标

- **新增侧边栏入口**：`🏪 卡产品` 独立页面
- **保留售卡流程**：售卡时仍从此页面选择卡产品
- **选择日期 + 截止日期展示**：新增卡产品时保留选择开卡日期的能力，已售列表展示截止日期

### 2.3 卡产品字段表

| 字段 | 类型 | 必填 | 说明 |
|------|:----:|:----:|------|
| 卡产品编号 | string | ✅ | 自动生成（CP+日期+序号） |
| 卡名称 | string | ✅ | 如"10次卡""月卡" |
| 卡类型 | string | ✅ | 次卡/期限卡/现金卡 |
| 标准售价 | number | ✅ | 产品的标准定价 |
| 总次数 | number | 次卡 | 次卡专属 |
| 赠送次数 | number | 次卡可选 | 新增字段 |
| 有效天数 | number | 次卡/期限卡 | 有效期天数 |
| 面值 | number | 现金卡 | 新增：仅展示，不关联售价 |
| 状态 | string | ✅ | 上架/下架 |
| 创建日期 | date | ✅ | 自动 |
| 备注 | string | | |

### 2.5 售卡对话框改进

在主流程中（会员管理 → 售卡），保留以下增强：

## 三、现金卡：面值与售价分离

### 3.1 现状

目前现金卡的字段：`售价` = `卡内余额`，没有独立的"面值"概念。

### 3.2 改造方案

**新字段设计**：

| 字段 | 用途 | 关系 |
|------|------|------|
| **面值** (face_value) | 卡产品上展示的"卡面金额"，**仅展示** | 保存在卡产品模板中 |
| **售价** (price) | 会员实际支付金额 = **储值金额** | 售卡时填写，存入会籍卡记录 |
| **余额** (balance) | 已售卡当前可用余额 | 消费后更新 |

**核心逻辑**：
- 卡产品模板：面值 ≠ 售价（可不同）
- 已售记录：售价 = 储值金额，余额 = 还剩多少钱可花
- 已售记录展示"面值"信息（从卡产品关联获取）

### 3.3 数据流

```
卡产品模板（is_product=1）：
  face_value = 500（展示用）
  price = 400（储值金额 = 优惠价售出）

  ↓ 售卡时

已售会籍记录（is_product=0）：
  price = 400（实付=储值金额）
  balance = 400（初始余额=售价）
```

## 四、次卡赠送次数

### 4.1 方案

| 级别 | 字段 | 说明 |
|:----:|------|------|
| **卡产品** | `赠送次数` | 新增字段，产品模板中定义 |
| **已售卡** | `赠送次数` | 新增字段，售卡时继承（或手动输入） |
| **已售表格** | 展示列 | 在"总次数"旁增加"赠送"列 |

### 4.2 展示效果

```
次卡已售数据表格：
┌─────┬──────┬──────┬──────┬──────┬──────┬──────┐
│编号 │ 会员 │ 类型 │ 总次数│ 赠送 │ 售价  │ 截止  │
├─────┼──────┼──────┼──────┼──────┼──────┼──────┤
│MC001│张三  │次卡  │ 10   │ +2   │¥800  │2027-05│
│MC002│李四  │次卡  │ 20   │ +0   │¥1500 │2027-06│
└─────┴──────┴──────┴──────┴──────┴──────┴──────┘
```

### 4.3 业务逻辑

- 总次数 = 购买次数 + 赠送次数（仅展示，不影响扣减逻辑）
- 消耗时只扣"购买次数"部分，赠送次数消耗完再扣实购
- 或者统一扣减：总次数 = 购买次数 + 赠送次数，全部消耗完为止

**推荐方案**：总次数 = 购买次数 + 赠送次数，统一扣减（简化逻辑）

## 十、售卡流程完整交互

```
会员管理 → 选中会员 → 点击"售卡"按钮
                    ↓
              弹出售卡对话框
                    ↓
              ① 自动带入会员信息（编号/姓名）
              ② 选择卡产品（下拉列表，加载产品目录）
              ③ 选中后自动填入：
                 - 卡名称、卡类型
                 - 标准售价（可修改）
                 - 总次数/有效天数/面值
              ④ 选择开卡日期（默认今天）
              ⑤ 自动计算截止日期并展示
              ⑥ 次卡输入赠送次数（默认0）
              ⑦ 确认 → 售卡成功
                    ↓
              已售会籍中新增记录
              展示：面值（现金卡）/赠送次数（次卡）/截止日期
```
