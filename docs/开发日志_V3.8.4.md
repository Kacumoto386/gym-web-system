# 开发日志 V3.8.4 — 导入结果下载（成功/失败数据导出 XLSX）

**日期**: 2026-05-25  
**类型**: feat

---

## 变更内容

### 1. 新增导入结果导出端点（`GET /api/import/{task_id}/export`）

**改动**：
- 根据 `ImportTask` 的 `preview_data`（全部数据行）和 `errors`（错误详情）分离成功/失败数据
- 错误行索引映射：`error.row`（Excel 行号）= 数据索引 + 2
- 批处理失败（如会员重复）时所有行归入失败 sheet，附加 batch 错误信息

**生成 XLSX 文件结构**：

**Sheet 1「成功导入」**：
- 中文表头行（蓝色加粗）
- 成功导入的全部数据行

**Sheet 2「导入失败」**：
- 中文表头 + "错误信息" 附加列
- 失败数据行 + 对应的错误详情
- 无失败记录时显示"无失败记录"

**文件命名**：`import_result_{task_id}.xlsx`（ASCII safe，浏览器兼容）

### 2. 历史表格添加下载按钮

**改动**：
- `import_history_table()` 操作列逻辑扩展：
  - 状态 `completed` / `failed` → 显示绿色 `下载结果` 链接
  - 状态 `confirming` → 保持现有 `执行导入` 按钮
  - 其他状态 → `-`
- 使用 `<a>` 标签直链下载（浏览器直接触发，无需 HTMX）

### 3. 支持所有导入类型

- 会员、员工、会籍卡、私教课 4 种导入类型均支持结果导出
- 字段映射复用 `IMPORT_FIELDS` 配置，保证中文表头一致性
