# 开发日志 V3.7.2 — 数据导出升级（CSV/XLSX/ZIP 批量导出）

**日期**: 2026-05-22  
**类型**: feature

---

## 变更内容

### 1. Excel（XLSX）格式导出

- 引入 `openpyxl` 库，支持导出为 `.xlsx` 格式
- 自动设置表头样式：加粗字体 + 浅蓝色背景
- 列宽自适应内容
- 日期列自动格式化为 YYYY-MM-DD

### 2. 多表打包 ZIP 下载

- 支持同时选择多张表，打包为一个 ZIP 压缩包下载
- ZIP 内每个表为一个独立 XLSX 文件

### 3. 日期范围筛选

- 每个表可配置主日期列（`DATE_COLUMNS` 映射表）
- 导出时支持按日期范围过滤数据
- 减少导出数据量，提高导出效率

### 4. 字段选择

- 支持选择需要导出的列（不选择则导出全部列）
- 前端提供列多选 UI

### 5. 导出接口重构

- `GET /api/export/table-list` — 返回可导出的表列表
- `GET /api/export/columns/{table_name}` — 返回指定表的列信息
- `POST /api/export/data` — 批量导出数据（JSON 请求体指定表/格式/日期/字段）
- `GET /api/export/download/{task_id}` — 下载生成的文件
- 响应使用 `StreamingResponse` 流式传输大文件

### 6. 前端导出页面重构

- 表选择 → 格式选择 → 日期筛选 → 字段选择 → 导出，分步操作
- 显示导出进度状态
- 导出历史记录
