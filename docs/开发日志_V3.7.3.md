# 开发日志 V3.7.3 — 课程包管理增强（包月管理）

**日期**: 2026-05-22  
**类型**: feature

---

## 变更内容

### 1. 包月管理（Tab 3）

- 新增 `MonthlyPass` CRUD 完整管理功能
- `GET /api/packages/monthly-passes/table` — 包月记录表格 HTML
- `POST /api/packages/monthly-passes` — 创建包月记录
- `PUT /api/packages/monthly-passes/{pass_id}` — 编辑包月记录
- `DELETE /api/packages/monthly-passes/{pass_id}` — 删除包月记录

### 2. 包月表格展示

- 显示包月编号、会员名、套餐名、类型（团课/私教）、包含课程、价格、有效期、状态
- 有效期智能显示：正常天数（灰色）、即将到期7天内（橙色）、今日到期（橙色高亮）、已过期（红色）
- 状态标签：正常（绿色）/ 已过期（红色）

### 3. 包月表单增强

- 类型切换：团课包月 / 私教包月
- 自动关联课程下拉选择
- 有效期自动计算（起止日期）
